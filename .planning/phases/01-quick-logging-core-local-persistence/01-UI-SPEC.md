## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, helper inline spacing |
| sm | 8px | Compact row spacing in set rows |
| md | 16px | Default field spacing and card inner padding |
| lg | 24px | Section padding in quick-log panels |
| xl | 32px | Gap between exercise sections |
| 2xl | 48px | Major separation between page blocks |
| 3xl | 64px | Top-level page breathing space |

Exceptions: minimum tap target 44px for icon-only actions (edit/delete set), while spacing still follows 4px multiples.

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 | 1.5 |
| Label | 14px | 600 | 1.4 |
| Heading | 20px | 600 | 1.2 |
| Display | 28px | 600 | 1.2 |

Contract limits: exactly 4 sizes (`14/16/20/28`) and exactly 2 weights (`400/600`) for this phase.

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `--background` (`oklch(1 0 0)` light, `oklch(0.145 0 0)` dark) | Page background and base surfaces |
| Secondary (30%) | `--card` / `--secondary` | Cards, grouped exercise blocks, secondary panels |
| Accent (10%) | `--primary` | Primary CTA, focused active input states, selected quick action |
| Destructive | `--destructive` | Delete set and destructive confirmations only |

Accent reserved for: `Добавить подход` CTA, active exercise selection state, inline "Сохранить изменения" action after editing a set.

---

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Добавить подход |
| Empty state heading | Подходов пока нет |
| Empty state body | Добавьте первый подход с весом и повторениями, чтобы начать тренировку. |
| Error state | Не удалось сохранить подход локально. Проверьте доступное место в браузере и повторите действие. |
| Destructive confirmation | Удалить подход: Удалить этот подход? Действие нельзя отменить. |

Interaction copy notes:
- Edit action label: `Изменить подход`
- Persisted suggestion label (LOG-04): `Предыдущий: {вес} кг × {повторы}`
